# E-commerce - TPI Prog3 - FRONTEND

## Descripción
Proyecto de e-commerce desarrollado para el Trabajo Práctico Integrador de Programación 3. La aplicación cuenta con catálogo de productos, carrito de compras, gestión de usuarios y panel de administración.

## Tecnologías
- TypeScript
- Vite

## Instalación
1. Abrir una terminal en la raíz del proyecto (`frontend/`).
2. Instalar dependencias:
   ```bash
